# modelisation-d-une-circulation-routi-re-en-2D-
Simulation 2D de trafic routier développée en Python avec Pygame. Le projet modélise un réseau urbain en grille où circulent véhicules autonomes, piétons et feux dynamiques. Les agents intègrent des mécanismes d’IA leur permettant de prendre des décisions en temps réel (priorités, évitement d’obstacles, optimisation du flux).
